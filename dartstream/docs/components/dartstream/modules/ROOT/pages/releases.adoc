== v0.0.2

=== Overview
Framework boundary and extension contract release for the DartStream OSS backend meta-package.

=== Highlights
* Added provider-neutral AI extension contracts for AI providers, requests, responses, and manager orchestration.
* Added provider-neutral ORM extension contracts for adapters, repositories, queries, and schema metadata.
* Added the backend `ds_dartstream` umbrella library export for framework consumers.
* Added documentation for OSS/SaaS boundaries, package maturity, frontend support, feature flags, AI extensions, and ORM integration.

=== Improvements
* Clarified that DartStream SaaS is a separate product built from the OSS framework base.
* Documented IntelliToggle and flagd as the supported feature flag lanes.
* Documented Drift as the first ORM adapter candidate while keeping concrete adapters out of the base package.
* Made Aortem open-source support explicit in the framework documentation.

=== Fixes
* Removed stale duplicate test and middleware artifact files from the touched framework areas.
* Tightened touched middleware CORS/context behavior covered by focused tests.

== v0.0.1

=== Overview
First main OSS release of the DartStream backend meta-package and workspace tooling.

=== Highlights
* Standard engine and extensions umbrella exports with refreshed registry manifests.
* Auth provider extensions for Auth0, Cognito, Entra ID, Firebase, Magic, Okta, Stytch, Transmit, Fingerprint, and Ping Identity.
* Persistence providers for PostgreSQL, MongoDB, Firestore, AWS S3-compatible storage, GCP Storage, OTLP logging, and Sentry logging.
* Reactive dataflow modules including message broker base with GCP Pub/Sub and WebSocket base with Socket.IO.

=== Improvements
* CLI tooling for init, setup, discovery, enable or disable, generate, and validate.
* Registry tooling scripts and a dev server example.
* Configuration templates for development and production middleware.

=== Fixes
* Resolved self-export loops in umbrella exports and removed placeholder configs.
* Added missing `_discoveryapis_commons` dependency for the GCP storage provider.

=== Notes
This is the first stable public release of the DartStream backend packages.
