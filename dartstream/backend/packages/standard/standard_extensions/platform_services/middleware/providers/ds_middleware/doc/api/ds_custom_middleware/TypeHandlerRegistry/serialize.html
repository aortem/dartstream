Serializes a value using a registered handler if available. Returns the value as-is if no handler is found.

Implementation

static dynamic serialize(dynamic value) {
  if (value == null) return null;
  var handler = _handlers[value.runtimeType];
  if (handler != null) {
    return handler.serialize(value);
  }
  return value;
}
